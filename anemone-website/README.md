## How to Use

1. **Open in Browser**

   - No server or build process required

2. **Customize Content**
   - Edit team member names and roles in the Team section
   - Update project details in the Projects section
   - Modify company information in the About and Footer sections

3. **Add Your Own Content**
   - Replace company motto and descriptions
   - Update social media links

## Color Scheme

The website uses a modern gradient color scheme:
- **Primary** - `#667eea` (Purple Blue)
- **Secondary** - `#764ba2` (Deep Purple)

## CSS Variables Reference

Edit these in `styles.css` to customize the website:

```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f093fb;
    --text-dark: #2d3436;
    --text-light: #636e72;
    --bg-light: #f8f9fa;
    --bg-white: #ffffff;
}
```

## Customization Guide

### Change Company Name
1. Update logo in navbar: `<div class="nav-logo">`
2. Update all references throughout the HTML

### Update Team Members
1. Edit HTML in the Team section
2. Modify avatar colors in CSS (`.avatar-1`, `.avatar-2`, etc.)
3. Add real images by replacing avatar divs with img tags

### Modify Color Scheme
1. Update CSS variables in `styles.css`
2. All gradient colors will automatically update

### Add Contact Forms
Add form elements in the footer contact section and connect with your backend service.

### Enable Social Links
Update href attributes in social links sections to point to actual social media profiles.
