### Step 3: Customization

#### Change Colors (Easiest)
1. Open `styles.css`
2. Find `:root { }` at the top
3. Modify these values:
   ```css
   --primary-color: #667eea;      /* Change to your brand color */
   --secondary-color: #764ba2;    /* Change to complement color */
   --accent-color: #f093fb;       /* Change accent color */
   ```

#### Update Company Information
1. Open `index.html`
2. Search for text you want to change:
   - Company description in About section
   - Team member names and roles
   - Project titles and descriptions
   - Office locations and contacts
3. Make your changes and save

#### Customize Team Members
1. Find the "Meet Our Visionaries" section
2. Update names and roles:
   ```html
   <h3>Your Name Here</h3>
   <p class="role">Your Title Here</p>
   <p class="bio">Your bio here</p>
   ```

#### Add Social Links
1. Find social links in footer
2. Update the `href` attributes:
   ```html
   <a href="https://linkedin.com/your-profile">
   ```

### Troubleshooting

**Q: Page looks broken**
A: Make sure all files (index.html, styles.css, script.js) are in the same folder

**Q: Animations not working**
A: Make sure JavaScript is enabled in your browser

**Q: Mobile menu not working**
A: Check browser console (F12) for errors

**Q: Colors not changing**
A: Clear browser cache (Ctrl+F5) after editing CSS
